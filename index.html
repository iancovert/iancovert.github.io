Research focus

My primary research focus is transparent machine learning. This involves reasoning about the information communicated by our data and how different features affect our models. I'm interested in efficient algorithms for large-scale models, theoretical foundations in game theory and information theory, and leveraging these ideas to improve how models are trained.

Some research threads in this area include:

  • Unified classes of explanation algorithms: many explanation approaches have been developed in the last decade, but it's often unclear when one method is preferable over another. Fortunately, it turns out that many algorithms are based on the same underlying principles [CovLunLee 2021] [CovLunLee 2020]
  • Efficient Shapley value estimation: the game-theoretic Shapley value is a compelling approach for model explanation, but its computation is a key obstacle for large models. Efficient approximations are now available due to improved statistical estimators, and a new approach based on deep learning [CovKimLee 2022] [CheCovLunLee 2022] [JetSudCovLeeRan 2021] [CovLee 2021]
  • Neural network feature selection: many uses of ML in biology and medicine can benefit from a reduced feature budget. We've developed techniques to enable feature selection for deep learning models in active, unsupervised and temporal settings [CovQiuLuKimWhiLee 2023] [CovGalWanSvoSumLee 2022] [TanCovFotShoFox 2021]
